# Prueba Tulen

_Servicio REST_


### Componentes utilizados 📋

```
flask
flask_restful

```

### Instalación 🔧

``` bash
# clone the repo
$ git clone https://github.com/jvergnol/test_rest.git

# go into app's directory
$ cd test_rest

# install app's dependencies
$ pip3 install flask
$ pip3 flask_restful
```
## Deployment 📦

``` bash
$ python3 api.py
```
